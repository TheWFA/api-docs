# List Competitions

Fetch a list of active competitions with their associated season data.

## üßæ Endpoint

```http
GET /competitions
```

**Base URL:** `https://api.thewfa.org.uk/competitions`

## üì• Query Parameters

This endpoint supports **pagination** through the standard `baseListQuerySchema`.

| Parameter       | Type    | Required | Description                         |
|----------------|---------|----------|-------------------------------------|
| `page`         | number  | No       | Page number (0-indexed). Default: 0 |
| `itemsPerPage` | number  | No       | Items per page. Default: 20. Min: 5, Max: 500 |

Example:

```http
GET /competitions?page=1&itemsPerPage=10
```
