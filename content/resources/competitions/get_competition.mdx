# Get Competition

Retrieve details of a single competition by its ID, including the currently active season and any historical names.


## üßæ Endpoint

```http
GET /competitions/:id
```

**Base URL:** `https://api.thewfa.org.uk/competitions/:id`

## üî¢ Path Parameters

| Parameter | Type   | Required | Description            |
|-----------|--------|----------|------------------------|
|  `id`     | string | Yes      | The ID of the competition to fetch |


## ‚úÖ Response

### Type

```ts
type Competition = {
  id: string;
  name: string;
  type: string;
  logo: string | null;
  activeSeason: {
    id: string;
    name: string;
    startDate: string;   // ISO 8601 date
    finishDate: string;  // ISO 8601 date
  };
  previousNames: {
    name: string;
    created_at: string;  // ISO 8601 date
  }[];
};
```

### Example

```json
{
  "error": false,
  "data": {
    "id": "prem",
    "name": "WFA National Premiership",
    "type": "league",
    "logo": "https://cdn.thewfa.org.uk/logos/prem.png",
    "activeSeason": {
      "id": "2024",
      "name": "2024/25",
      "startDate": "2024-08-01T00:00:00Z",
      "finishDate": "2025-06-01T00:00:00Z"
    },
    "previousNames": [
      {
        "name": "National Championship",
        "created_at": "2018-06-01T00:00:00Z"
      }
    ]
  }
}
```

## ‚ö†Ô∏è Error Responses

| Code | Message                 | Cause                                |
|------|-------------------------|--------------------------------------|
| 400  | Invalid ID              | Missing or malformed \`id\` parameter |
| 404  | Competition not found   | No competition found with the given ID |
| 500  | Internal Server Error   | Unexpected server error or crash     |


## üìù Notes

- This endpoint also returns a list of all previous names the competition has had, in chronological order based on `created_at`.
- When displaying a competition in a historical context, please use the appropriate name based on the name history.
