# Get Match

Retrieve full match details, including team lineups and a chronological list of match events.

## üßæ Endpoint

```http
GET /matches/:id
```

Base URL: `https://api.thewfa.org.uk/matches/:id`

## üî¢ Path Parameters

| Parameter       | Type     | Required | Description                         |
|-----------------|----------|----------|-------------------------------------|
| `id`            | string   | Yes      | The ID of the match to retrieve     |

## ‚ö†Ô∏è Error Responses
| Code | Message               | Cause                                  |
|------|-----------------------|----------------------------------------|
| 400  | Invalid ID            | Missing or malformed `id` parameter    |
| 404  | Match not found       | No match found with the given ID       |
| 500  | Internal Server Error | Unexpected server error                |

## üìù Notes
- Lineups are split by team, based on the team ID in the match data.
- Events are ordered by match time and period.
- If a player does not appear in the match, `position` may be `null`.
- Events with type `goal` include penalty and own-goal information.
